- Tạo 1 project trên GitHub
1. Tạo 1 GitHub repository (folder) - nơi chứa code
2. Clone project này: git clone link_htpps
3. Viết code: ( trên git bash )
 git status (kiểm tra trạng thái files)
 git add (thêm file)
 git commit -m "your message" (tạo nội dung commit)
 git push origin main


- Tạo project local và đẩy lên GitHub
1. Tạo 1 GitHub repo
2. git init
3. Viết code:
 git status (kiểm tra trạng thái files)
 git add (thêm file)
 git commit -m "your message" (tạo nội dung commit)
 git remote add origin htpp://.....
 git push origin master
